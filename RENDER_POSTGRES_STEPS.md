# FleetAI v0.2 — PostgreSQL READY

Эта версия готова для Render + PostgreSQL.

## Что внутри

- Flask-сайт FleetAI
- SQLAlchemy
- PostgreSQL через переменную `DATABASE_URL`
- Доходы
- Ремонты
- Дополнительные вложения в авто
- Вложения инвесторов
- Выплаты инвесторам
- Расчет инвесторов по периоду с 15-го по 15-е
- Автоматическое создание таблиц при запуске

## Что должно быть в Render Environment

В Web Service `fleetai` должна быть переменная:

```text
DATABASE_URL=postgresql://...
```

Можно вставлять Internal Database URL из Render.

## Как обновить GitHub

1. Распакуй архив.
2. Открой свой репозиторий `edgarfleetai/fleetai`.
3. Нажми **Add file → Upload files**.
4. Загрузи все файлы из распакованной папки с заменой старых.
5. Нажми **Commit changes**.

## Как задеплоить

1. Открой Render → Web Service `fleetai`.
2. Нажми **Manual Deploy → Deploy latest commit**.
3. Дождись статуса **Live**.

## Проверка

На сайте введи:

```text
665 получил 1000
```

Потом:

```text
665 доп ГБО 35000р
```

Потом обнови страницу. Записи должны остаться.

Для полной проверки сделай еще один Manual Deploy. Если записи остались — PostgreSQL подключен правильно.

## Важно

Если раньше ты присылал строку подключения в чат, после настройки лучше сменить пароль базы в Render.
