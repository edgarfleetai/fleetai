# FleetAI Cloud Investor Periods

Добавлено:
- расчетный период инвесторов 15-го по 15-е;
- автоматический расчет дохода, расходов, прибыли;
- расчет суммы к выплате инвестору по проценту машины;
- кнопка закрытия периода;
- расходы типа `owner_only`, если написать в сообщении "не делить" или "моя ответственность".

Deploy:
```bash
gunicorn app:app
```


Добавлено в этой версии:
- дополнительные вложения в машину: `665 вложение ГБО 58000`;
- вложения инвестора в конкретную машину: `665 инвестор Иван вложил 300000 50%`;
- выплаты инвестору: `665 выплата Ивану 25000`;
- в таблице машин считается реальная стоимость: цена покупки + доп. вложения.
