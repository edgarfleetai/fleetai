# FleetAI Cloud Investor Periods

Добавлено:
- расчетный период инвесторов 15-го по 15-е;
- автоматический расчет дохода, расходов, прибыли;
- расчет суммы к выплате инвестору по проценту машины;
- кнопка закрытия периода;
- расходы типа `owner_only`, если написать в сообщении "не делить" или "моя ответственность".

Deploy:
```bash
gunicorn app:app
```
